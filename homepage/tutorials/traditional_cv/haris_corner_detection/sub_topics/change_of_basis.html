Change of Basis

Any vector in a vector space of a finite dimension \large 'n' can be represented uniquely via set of \large 'n' basis vectors and \large 'n' scalars known as coordinates. An example of this is shown below with respect to a \large 2Dcartesian coordinate system, Let's say we have a vector   \large [2, 3]^{T} which can be represented using the basis vectors \large (\bar{x}, \bar{y}) of the cartesian coordinate system as,

\large \bar{x} = \begin{bmatrix} 1 \\ 0 \end{bmatrix} \hspace{0.20cm} \bar{y} = \begin{bmatrix} 0\\ 1 \end{bmatrix}
\large \begin{bmatrix} 2 \\ 3 \end{bmatrix} = 2 \begin{bmatrix} 1 \\ 0 \end{bmatrix} + 3 \begin{bmatrix} 0 \\ 1 \end{bmatrix}


As we know a set of vectors qualify has basis vectors only when the set is able to span the entire vector space and the vectors in the set are linearly independent and zero vector is part of the set. A change of basis is a process of converting a coordinate relative to one basis vectors to another set of basis vectors. Mathematically, this can be represented as,

\large [\overrightarrow{a}]_{S} = C_{S} [\overrightarrow{a}]_{M}

where \large S is the standard basis/coordinate system where as \large M is an another coordinate system. So, to convert a vector \large \overrightarrow{a} from the \large M's coordinate system to \large S's coordinate system. One should perform a change of basis which is represented as \large C_{S} meaning as the basis vectors of \large M's coordinate system in \large S's coordinate system. The columns of \large C's is represented as,

\large M = \{ \overrightarrow{v_{1}}, \overrightarrow{v_{2}}, ....., \overrightarrow{v_{k}} \}

\large C_{S} = \begin{bmatrix} . & . & ..... & . \\ . & . & ..... & . \\ \overrightarrow{v_{1}} & \overrightarrow{v_{2}} & ..... & \overrightarrow{v_{k}}\\ . & . & ..... & . \\ . & . & ..... & . \\ \end{bmatrix}

Here \large k represents the dimension of the vector space \large M and each vector \large \overrightarrow{v}_k in the set belongs to the space \large \mathbb{R}^{n} where \large n is the dimension of the vector space \large S. Now let's do the reverse, how to get vector \large \overrightarrow{d} in M's coordinate system given \large C's and \large [\overrightarrow{d}]_{S}? One can solve this systme of linear equation by converting \large C's to reduced row echolan form \large rref(C) if and only if \large \overrightarrow{d} falls in the span of the basis vectors of \large M. There is one more condition that the column vectors of \large C are linearly independent which in this cases is statisfied because the column vectors are the basis vectors of M's coordinate system and as we know for a set of vectors to be a basis vectors the vectors in the set must be linearly independent. In the next section we will look at another way to compute this if a special condition is satisfied.

Invertiable Change of Basis

Let's continue with the vector \large \overrightarrow{d} and the vector spaces \large S & \large M. As above let \large C be a change of basis matrix, then to represent the vector \large \overrightarrow{d} in \large S coordinate system from \large B coordinate system one must do the following operation given below,

\large [\overrightarrow{d}]_{S} = C_{S} [\overrightarrow{d}]_{M}

Now let's ponder over the question "How to get vector \large \overrightarrow{d} in M's coordinate system given \large C's and \large [\overrightarrow{d}]_{S}?". As mention before we can treat this as a system of linear equation and solve for \large [\overrightarrow{d}]_{M}. But, what happens if \large C is invertiable that is it stasfies two condiation given below,

  1. \large C is a square matrix.
  2. The column vectors in \large C are linearly independent.

If \large C is a square matrix then, \large k = n stating that the basis vector set of \large M which are \large \{ \overrightarrow{v_{1}}, \overrightarrow{v_{2}}, ....., \overrightarrow{v_{k}} \} \subset \mathbb{R}^{n} and has stated earlier the column vectors of \large C are linearly independent because the columns are the basis vectors. Then \large M is also an basis set for \large \mathbb{R}^{n}. Thus proving that if \large C is invertiable, then span of \large M equals \large \mathbb{R}^{n} or we can rephrase this as if span of \large M is \large \mathbb{R}^{n} then \large C is invertiable. Hence giving as the following equation below,

\large [\overrightarrow{d}]_{M} = C_{S}^{-1} [\overrightarrow{d}]_{S}

This shows a vector space can have more than one basis set to span it. In the next section we will look into how to apply the change of basis to an transformation matrix.

Transformation Matrix with Respect To A Basis

Let's say some linear transformation \large T that is mapping from \large \mathbb{R}^{n} to \large \mathbb{R}^{n} in the the standard coordinate system \large S and if \large T is applied to a vector \large \overrightarrow{x} which is also in the standard coordinate system then using the transformation matrix \large A we can obtain \large T(\overrightarrow{x}) as shown below,

\large T: \mathbb{R}^{n} \to \mathbb{R}^{n} \\ \\

\large T(\overrightarrow{x}) = A\overrightarrow{x}

Now we want to apply the same transformation on the same vector \large \overrightarrow{x} but all of this will be happening in a different coordinate system i.e. different set of basis vectors which are \large B = \{ \overrightarrow{v_{1}}, \overrightarrow{v_{2}}, ....., \overrightarrow{v_{k}} \} will the span the space \large \mathbb{R}^{n}. The transformation matrix \large A in this space is given by \large D. Thus giving us the equation below,

\large [T(\overrightarrow{x})]_{B} = D[\overrightarrow{x}]_{B}

\large D is a transformation matrix for \large T with respect to the basis set B. As \large D and \large B are the same transformation but with respect to different set of basis vectors then there should be some relationship between these two, in the upcoming content we will be deriving this relationship between the two.

Relationship between D and A:

The change of basis vector which can be generated from B is represented as C of dimension n \times n where \large C is a square matrix and the columns of \large C are linearly independent thus making \large C invertible. Hence, the following equations below are valid for a vector \large \overrightarrow{x},

\large \overrightarrow{x} = C [\overrightarrow{x}]_{B}

\large [\overrightarrow{x}]_{B} = C^{-1} \overrightarrow{x}

Using the equations defined in this section we can write \large D in terms of \large A as,

\large D[\overrightarrow{x}]_{B} = [T(\overrightarrow{x})]_{B} = [A\overrightarrow{x}]_{B} = C^{-1}A \overrightarrow{x} = C^{-1}AC[\overrightarrow{x}]_{B}

\large D = C^{-1}AC

With simple matrix arithematic we can rewrite the above equation as \large A = C D C^{-1}. This will be extended to eigen decomposition or diagonalization which inturn help in understanding one of the application of Eigen values and eigen vectors in Ellipse.
