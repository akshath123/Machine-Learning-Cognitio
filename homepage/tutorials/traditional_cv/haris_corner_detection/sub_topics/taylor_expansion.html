Taylor Expansion

Taylor expansion approximates a function using a polynomials. To understand Taylor expansion I have broken down this tutorial into three sections which are Higher order derivatives, Maclaurian series and Taylor expansion. Here, I hope the reader to develop a geometeric intution behind Taylor series which we will use later to derive other awesome stuff.

I. Higher Order Derivatives

As we know, a derivative or gradient of a function with a variable \large 'x' measures the rate of change of the function with respect to the change in \large 'x'. The previous statement is also known as first derivative test and can be represented as,

\large derivative \hspace{2mm} of \hspace{2mm} the \hspace{2mm} function \hspace{2mm} f(x) = f'(x)

Now, what does the second derivative tell us? One can think this as the rate of change of rate of change or change to slope. The \large n^{th} derivative of the function is basically the rate of change of   \large (n - 1)^{th} derivative. Which can be represented as,

\large n^{th} \hspace{2mm} derivative \hspace{2mm} of \hspace{2mm} the \hspace{2mm} function \hspace{2mm} f(x) = f^{n}(x)

II. Maclaurian Series

Given a function \large f(x) we can approximate the function around zero using a polynomial \large P(x)with the assumption that the value the function takes at zero is known similarly the value the higher orderivates take at zero is also known (or can be obtained). From here let's derive the maclaurian series,

As mentioned earlier we know the value that the function takes at zero, so to approximate the function using the polynomial then the polynomial must take on the same value the function takes at zero. This is illustrated below,

\large P(0) = f(0) \implies P(x) = f(0)

Above mentioned polynomial approximates the function only at zero but not around it, to get a better approximation we keep adding polynomials to \large P(x) such that it equates higher order derivatives of the function \large f(x) without changing the other order polynomials in \large P(x). To understand this we will approximate the function upto the third derivative and then write the generalized version of the maclaurian series.

First Derivative approximation

So, as we would like   \large P(0) = f(0)   we would also like the polynomials first derivative equal the first derivative of the function   \large P'(0) = f'(0). To this we add the first degree polynomial such that the already constructed polynomial remains unfazed with this addition to the polynomial as shown below,

\large P(x) = f(0) + xf'(0)

Now let's check if the above polynomial approximates the function and it's first derivative at the value zero.

\large P(0) = f(0) + (0)f'(0) = f(0)

\large P'(x) = \cancelto{0}{f(0)} + \cancelto{1}{x}f'(0) = f'(0)

Now we can say the polynomial approximates the funcation at zero as well as it's first derivative (just around it). Similarly, let's try to increase the radius of the approximation or make the approximation done by the polynomial even better. For that, we follow the same procedure as above for higher order derivatives.

Second Derivative approximation

As above we want the function's second derivative equal the second derivative of the polynomial at the value zero. This can demonstrated by adding a second degree polynomial to  \large P(x) as shown below,

\large P(x) = f(0) + xf'(0) + \dfrac{1}{2}x^{2}f''(0)

If you are wondering how did   \large \dfrac{1}{2}   come, it is basically for making the second order derivative of the polynomial constant free. Let's first check previous approximation holds good at the value zero.

\large P(0) = f(0) + (0)f'(0) + \dfrac{1}{2}(0)^{2}f''(0) = f(0)
\large P'(x) = \cancelto{0}{f(0)} + \cancelto{1}{(x)}f'(0) + \cancelto{1}{\dfrac{2}{2}} (x)f''(0)
\large P'(0) = 0 + f'(0) + (0)f''(0) = f'(0)

So, from the above equation we can see the previous approximation is unfazed by addition of the second degree polynomial. The equation below shows how the second order derivative of the polynomial approximates the function's second order derivative.

\large P''(x) = \cancelto{0}{f(0)} + \cancelto{0}{xf'(0)} + \cancelto{1}{\dfrac{2}{2}x}f''(0) = f''(0)

Third Derivative approximation

Similarly let's find the third order approximation of the function using the polynomial given below,

\large P(x) = f(0) + xf'(0) + \dfrac{1}{2} f''(0) + \dfrac{1}{6} f'''(0)

\large P(x) = \dfrac{1}{1}x^{0}f(0) + \dfrac{1}{1}x^{1} f'(0) + \dfrac{1}{1 \times 2}x^{2} f''(0) + \dfrac{1}{1 \times 2 \times 3} x^{3} f'''(0)

\large P(x) = \dfrac{1}{0!}x^{0}f(0) + \dfrac{1}{1!}x^{1} f'(0) + \dfrac{1}{2!}x^{2} f''(0) + \dfrac{1}{3!} x^{3} f'''(0)

As above you can prove that the addition of the thrid degree polynomial will not change the approximation of first degree and second degree polynomials. In general, if you keep adding higer degree polynomials then the approximation of the function becomes even better at the value zero.

Generalized Version of Maclaurian series

Finally, for the sake of completeness let's write the general equation of Maclaurian series which is shown below,

\large P(x) = \dfrac{x^{0}}{0!}f(0) + \dfrac{x^{1}}{1!} f'(0) + \dfrac{x^{2}}{2!}f''(0) + \dfrac{x^{3}}{3!} f'''(0) + ..... + \dfrac{x^{n}}{n!} f^{n}(x)

NOTE: this is the approximation of the function at the value zero and around it.

III. Taylor Series

If you have understood the Maclaurian series then it's pretty easy to understand Taylor series, the taylor series approximates the function at any arbitary value   \large 'z'  . We can derive the polynomial for this pretty much the same way has we derived the polynomial for maclaurian series,

Approximation at \large 'z' can be given as,

\large P(x) = f(z)

Similarly, the first order approximation of the function at the value \large 'z' can be given as,

\large P(x) = f(z) + f'(z)(x-z)

To verify if the approximation is valid we can follow the steps similar to the steps of Maclaurian series as shown below,

\large P(z) = f(z) + \cancelto{0}{f'(z)(z-z)} = f(z)

\large P(x) = f(z) + f'(z)(x-z)
\large P(x) = f(z) + xf'(z) - zf'(z)
\large P'(x) = \cancelto{0}{f(z)} + \cancelto{1}{x}f'(z) - \cancelto{0}{zf'(z)} = f'(z)

Now, I hope you understand the role of using \large (x-z) as a coefficient in the first degree polynomial. The same procedure can be done for higer order polynomials yielding better approximation of the function at the value \large 'z'. The generalized verion of taylor series is given below,

\large P(x) = \dfrac{(x-z)^{0}}{0!}f(z) + \dfrac{(x-z)^{1}}{1!}f'(z) + \dfrac{(x-z)^{2}}{2!}f''(z) .. +\dfrac{(x-z)^{n}}{n!}f^{n}(z)

NOTE: Not all funcations can be approximated using Taylor series, for example the function \large \ln(x). However, this is good tool because we can use this to derive or understand the functions behaviour at a arbitary value. Moreover, here we assumed the function depends on single variable but if the function depends on more than one variable then we use partial derivatives to define the Taylor and Maclaurian series.
